What's new in version 7.0-testing:
==============================
	[*] Minor bug fixes
	[+] Added the ability to stop the cleaning process (thread) by clicking again on the button used to initiate the cleaning process.
	[+] Added Android cleaners (using ADB), Can be used to clean up temporary data on your Android smartphone wirelessly or using a USB Cable.

	[+] Added BeamNG.Drive temp folder cleaning
	*-Please note that not all new features and bug fixes are listed here, and you may see other improvements/or new features that aren't listed here.
