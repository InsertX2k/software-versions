What's new in version 6.4-stable:
=================================
	[*] Now the Recycle bin cleaner erases the recycle bin of all drives associated to this PC.
	[*] Fixed the issue with Windows Update restart function.
	[*] Fixed issues with Android cache cleaner not working.
	[+] Added Offline Web pages cleaning.
	[+] Added donators window (can be accessed by pressing the "Donators" button from the "About" window)
	[*] Improvements for Arabic language translations.
	[*] Overall stability improvements.
	[*] Fixed major bugs with the Updater program.
	[+] Added Windows Component Store (WinSxS) cleaner. (This will help you free up a lot of disk space)
	[*] Fixed issues in Performance Logs cleaner.
	[+] Added Web browser history cleaning feature (it is a part of the Browser's cache cleaner)
	[+] Added Windows Service Pack cleaner. (This will help you free up a lot of disk space)
	[+] Added Windows.old folder cleaner. (This will help you free up a lot of disk space)
	[*] Improved the way Temp_Cleaner GUI handles system files permissions.
	[-] Removed Iconcache.db cleaner
	[-] Removed unnecessary warnings
	*-Please note that not all new features and bug fixes are listed here, and you may see other improvements/or new features that aren't listed here.
